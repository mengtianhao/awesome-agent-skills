# Awesome Agent Skills

[中文](README_zh.md)

A personal index of useful AI agent skills, skill repositories, plugins, and related toolkits.

This repository tracks where each skill comes from, what it does, how to install it, and whether it overlaps with skills already installed locally.

## Entry Format

Use [templates/skill-entry-template.md](templates/skill-entry-template.md) when adding a new skill or repository.

Each entry should include:

- repository URL
- official documentation URL, if different
- available skills
- installation notes
- overlap with locally installed skills
- why it is useful
- status and last checked date
