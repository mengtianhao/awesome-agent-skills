# Awesome Agent Skills

[English](README.md)

这是一个个人维护的 AI Agent 技能索引库，用来收藏有用的 skills、技能仓库、插件和相关工具集。

这个仓库会记录每个技能来源、用途、安装方式、主要能力，以及它和本地已安装技能之间是否存在重叠。

## 索引

| 名称 | 类型 | 主要用途 | 条目 |
| --- | --- | --- | --- |
| MiniMax Office Skills | 技能仓库 | Office 文档、前端/媒体工作流、移动端、Shader | [skills/minimax-office-skills.md](skills/minimax-office-skills.md) |
| PaperSpine | 技能套件 | 学术论文/报告写作、引用支持、LaTeX/PDF/Word 审计 | [skills/paper-spine_zh.md](skills/paper-spine_zh.md) |

## 分类

- Office 与文档处理
- 前端与全栈开发
- 媒体生成
- 移动端开发
- Shader 与图形工作流
- 科研与学术写作
- 自动化与效率工具

## 条目格式

新增技能或技能仓库时，可以使用 [templates/skill-entry-template.md](templates/skill-entry-template.md) 作为模板。

每个条目建议包含：

- 仓库地址
- 官方文档地址，如和仓库不同
- 可用技能列表
- 安装说明
- 与本地已安装技能的重叠情况
- 为什么值得保留
- 状态与最后检查日期

## 机器可读数据

结构化记录保存在 [data/skills.json](data/skills.json)。
